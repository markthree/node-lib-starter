# node-lib-starter

开箱即用的 `node` 库模板

<br />

## features 🦕

- pnpm 的
- vitest 测试
- 开箱即用的
- typescript 的

<br />
<br />

## Usage 🦖

### install

```shell
pnpm i
```

### test

```shell
pnpm test

# or pnpm test:watch
```

### build

```shell
pnpm build

# pnpm build:esm 仅打包 esmodule
# pnpm build:cjs 仅打包 commonjs
# pnpm build:stub 打包插桩
```

### coverage

```shell
pnpm coverage
```

### dev

```shell
pnpm dev
```

### release

```shell
pnpm release
```

### play

```shell
pnpm play
```
